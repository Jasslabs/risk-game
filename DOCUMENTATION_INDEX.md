### 🔧 Tools & Utilities

#### **diagnostics.html** (System Test)
- **Time**: 5 minutes
- **What**: Test your system
- **How**: Open in browser:
  ```
  http://localhost/aiprojectcopilot/diagnostics.html
  ```
- **What it checks**:
  - Browser compatibility
  - JavaScript files loading
  - Canvas support
  - Database connection
  - API connectivity
- **Use when**: Game won't load or save doesn't work

#### **setup.bat** (Windows Setup)
- **Time**: 1 minute
- **What**: Automates checking for XAMPP
- **How**: Double-click file
- **What it does**:
  - Checks if Apache is running
  - Checks if MySQL is running
  - Verifies all game files present
  - Shows setup steps
- **Use when**: First time setting up

## ✅ Quick Checklist
